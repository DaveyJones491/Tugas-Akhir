# Flask Web App Tutorial

## Setup & Installation

Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```

```bash
pip install -r requirements.txt
```

## Dependencies 
flask
Flask-SQLAlchemy
flask-login
datetime
base64
time
typing
werkzeug
pyaudio
wave
requests
playsound 1.2.2 (Must be 1.2.2 or will not work)

## Running The App

```bash
python main.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
